#Techsitter: The Ultimate Knowledge Guardian for Your Technical Support
Think of Techsitter as the "super-powered nanny" for your technical documentation. While you are busy solving high-level problems, Techsitter sits with your data, learns every detail, and stays ready to answer any question the moment a customer asks.

#Core Capabilities
Diverse Document Indexing: Securely load and index your primary documents, including text files, comma separated values, and Statement of Works.
Dynamic Knowledge Base: Treat your signed contracts and technical specifications as a living library that the agent references in real-time.
Live Web Integration: Designate specific URLs to allow the agent to pull from official online documentation or external Retrieval-Augmented Generation sources.
Single-Query Intelligence: Prompt the agent with a single customer question to receive a synthesized, accurate response based on your private data.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
